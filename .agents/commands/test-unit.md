ユニットテストのみを実行してください。

## 手順

以下のコマンドを実行する:
```
fvm dart run melos run test:unit
```

## 対象

`test/` ディレクトリが存在するすべての Flutter パッケージの `test/` 以下のテスト。
Golden テストや E2E テストは含まれません。

## 結果の報告

- ✅ 全テストパス: 「ユニットテスト全パス」と報告
- ❌ 失敗があった場合: 失敗したテストのファイルパス・テスト名・エラーメッセージを整理し、原因と修正案を提示する

## テスト実装ルール（確認用）

- 日時依存ロジックのテストでは `withClock()` を使用すること（`DateTime.now()` 直接参照は禁止）
- `_fixedNow` 等の定数はテストファイルのトップレベルに定義すること
