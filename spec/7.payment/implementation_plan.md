# 💳 決済・支払いカテゴリ クイズ一覧 (PayPay風統合版)

「文字が読めないキャッシュレス決済アプリ」を舞台に、現代の支払いUIセオリーを問うクイズシリーズ。すべてのクイズが「ホーム画面」から開始され、特徴的なボトムナビゲーションやタイルアイコンを駆使してミッションをクリアします。

## 概要

| 項目 | 内容 |
|------|------|
| パッケージ | `packages/quizzes/payment` |
| クイズ総数 | 4問 |
| テーマ | QR/バーコード決済アプリ（PayPay風） |
| UIの特徴 | 中央が大きく突出したボトムナビ、残高・支払い元カード（スワイプ対応）、タイル型機能アイコン |
| 共通フロー | すべてのクイズは「ホーム画面」からスタートし、目的に応じて画面やタブを遷移する |

---

## ベースUI設計要件（PayPay風）

1. **画面全体の構成とタブの挙動**:
   * 画面は大きく「メインコンテンツ領域」と下部の「ボトムナビゲーション」で構成される。
   * ホーム以外のタブ（履歴、ポイント、アカウント等）をタップした際は、メインコンテンツ領域がそれぞれのダミー画面に切り替わるように実装し、ひとつのアプリとしての体裁を保つ。
   * テキストはすべて異世界言語（カスタムフォント）を使用する。

2. **特徴的なボトムナビゲーション**:
   * 5つのボタンを配置し、**中央のボタンのみ大きく上部に突出（盛り上がった）デザイン**とする。
   * **左側2つ**: 🏠 ホーム（開始地点）、履歴・明細（ダミー画面）
   * **中央（巨大・突出）**: 💳 **支払い（※このボタンのみタブ切り替えではなく、支払い画面を開くショートカットとして動作）**
   * **右側2つ**: ポイント（ダミー画面）、アカウント設定（ダミー画面）

3. **ホーム画面（開始時のメインコンテンツ）**:
   * **上部（残高・支払いカード）**: 
     * 現在の残高額（数字のみ読める）と、その横に「目のアイコン（👁️）」。
     * その下（または横）に、**小さな支払い用バーコードと現在の支払い元（例：残高）**が表示されたエリアがあり、**左右スワイプ（カルーセル）で支払い元を切り替えられる**。
   * **中部（機能タイル）**: 4つの丸いアイコンを並べる。「スキャン（📷）」「送金（💸または🤝）」「チャージ（＋）」「その他（…）」など。

4. **支払い画面（中央の巨大ボタンタップ後）**:
   * 画面中央に決済用のバーコードとQRコードが表示される。
   * バーコードの下に「現在の支払い元（例：残高アイコン）」を示すボタンがあり、タップで支払い方法を変更できるUI。

### Quiz 1: バーコード提示クイズ
**ミッション**: レジに来ました。店員さんに支払い用のバーコードを提示してください。
* **難易度**: ★1
* **クリア条件**: 
  1. ホーム画面で、ボトムナビゲーションの**中央にある一番大きなボタン**をタップする。
  2. バーコード・QRコードが全画面に表示されればクリア。
* **Insight**: 「一番目立つ大きなボタン＝アプリの主目的（支払い）」という、決済アプリにおける最強のアフォーダンスを問う。

### Quiz 2: 残高非表示クイズ
**ミッション**: 誰かに画面を見られます！現在の「残高」を隠してください。
* **難易度**: ★2
* **クリア条件**: 
  1. ホーム画面上部の「残高表示カード」を見る。
  2. 残高の金額の横にある**「目のアイコン（👁️）」**をタップする。
  3. 金額が「***」に伏せ字になればクリア。
* **Insight**: 「目のアイコン＝表示/非表示の切り替え（パスワード等と同じ）」という普遍的なアイコンリテラシーを問う。

### Quiz 3: 送金クイズ
**ミッション**: 友達に5,000円を送金してください。
* **難易度**: ★3
* **クリア条件**: 
  1. ホーム画面中部の4つのタイルアイコンから**「送金（💸または握手🤝などのアイコン）」**をタップする。
  2. テンキー画面に遷移後、「5,000」と入力する。
  3. 画面下部の「送信（確定）」ボタンをタップすればクリア。
* **Insight**: 機能が並んだタイルの中から、アイコンの形状だけで「送金」という概念を見つけ出す直感力を問う。

### Quiz 4: 支払い元変更クイズ
**ミッション**: 残高が足りません！支払い方法を「クレジットカード」に変更してからバーコードを提示してください。
* **難易度**: ★4
* **クリア条件**: 以下の**いずれかのルート**で達成すること。
  * **【ルートA（支払い画面から変更）】**
    1. ボトムナビゲーション中央の大きなボタンをタップし、支払い画面を開く。
    2. バーコードの下にある「支払い元表示（例：残高アイコン）」のボタンをタップする。
    3. 下から出てきたボトムシートから「クレジットカード（💳）」を選択する。
  * **【ルートB（ホーム画面でスワイプ）】**
    1. ホーム画面上部にある「支払い元表示・ミニバーコード」のカードエリアを**左右にスワイプ**し、表示を「クレジットカード（💳）」に切り替える。
    2. ボトムナビゲーション中央の大きなボタンをタップし、支払い画面を開く。
* **Insight**: 「現在の設定値が書かれた場所はタップで変更できる」というドロップダウンのメンタルモデルと、「カード状のUIはスワイプで切り替えられる」という現代的なジェスチャーUIの理解を問う。
