# 🔣 NantoNack カスタム言語（異世界言語）仕様書

本アプリでは、ユーザーが「文字の意味」ではなく「UIの配置や形」から直感で操作を推測できるよう、画面上のテキストを専用のカスタム言語に変換して表示します。

## 📌 言語仕様（変換ルール）

1. **ベース言語**: 変換の基となるテキストはすべて「英語（English）」を使用します。
2. **アルファベット置換（完全ランダム）**:
   * `A〜Z` および `a〜z` の26文字を、母音・子音の区別なく完全にランダムな別のアルファベットに1対1で置換します（カオス感を演出するため）。
   * **大文字・小文字の維持**: 元のテキストがキャピタルケース（例: `Cart`）であれば、変換後も先頭は大文字（例: `Ukyb`）とし、元の単語の視覚的なパターンを維持します。
3. **数字置換（ギリシャ文字・完全ランダム）**:
   * `0〜9` の数字は、桁数を変えずに1対1で「小文字のギリシャ文字」に置換します。
   * 規則性を排除するため、アルファベット順（α, β, γ...）ではなく完全にランダムな順序で割り当てます。
4. **記号・スペースの維持**:
   * 空白（スペース）、カンマ（`,`）、ピリオド（`.`）、コロン（`:`）、通貨記号（`$` や `¥`）、ハイフン（`-`）などの各種記号は**一切変換せずそのまま表示**します。
   * これらはユーザーにとって「ここは価格だ」「ここは区切りだ」と認識させるための強烈なUIのヒント（アフォーダンス）となります。

### 💡 変換例

* テキスト: Add to Cart
  * 変換後: Kmm zs Ukfz （元のA, Cが大文字であることを維持）
* テキスト: Price: $1,980
  * 変換後: Ofkux: $γ,δζη （記号はそのままで、数字がランダムなギリシャ文字化される）
* テキスト: Next Step
  * 変換後: Jxtz Wzxo
