### Quiz 1: 現在地確認クイズ

**ミッション**: 地図で現在地を表示してください。

* **難易度**: ★1
* **クリア条件**: 現在地ボタン（📍）をタップし、現在地を地図上に表示すること。
* **不正解条件**: なし（現在地ボタン以外の操作は判定に影響しない）
* **Insight**: 現在地アイコン（ピン・矢印）は地図アプリの世界標準。コンパスアイコンは向きをリセットする別機能。

---

### Quiz 2: 学校情報表示クイズ

**ミッション**: 地図上の学校アイコンをタップして情報を表示してください。

* **難易度**: ★2
* **正解場所**: `p5` University（学校アイコン 🏫）
* **クリア条件**: 地図上の学校（University）アイコンをタップして詳細パネルを表示すること。
* **不正解条件**: 学校以外の場所（City Hall / Grand Park / Main Station / Shopping Mall）をタップ → 即時不正解
* **Insight**: アイコンで施設の種別を識別できるかが問われる。学校アイコンは世界共通で帽子・建物・鉛筆のデザインが多い。

---

### Quiz 3: ルート案内クイズ

**ミッション**: 公園まで電車でルートを案内してください。

* **難易度**: ★3
* **正解目的地**: `p2` Grand Park（公園アイコン 🌳）
* **正解交通手段**: インデックス 2（Transit / 電車 🚃）
* **クリア条件（3ステップ）**:
  1. 地図上の公園（Grand Park）をタップしてルートパネルを表示する
  2. 交通手段から電車（Transit）を選択する
  3. 「案内を開始」ボタンをタップする
* **不正解条件**:
  * 公園以外の場所をタップ → 即時不正解
  * 電車以外の交通手段（車・徒歩・自転車）を選択 → 即時不正解
* **Insight**: 車・徒歩・電車・自転車のアイコンは世界標準。目的地タップ→交通手段選択→開始という3ステップの流れが地図アプリの基本操作。

---

### Quiz 4: お気に入り保存クイズ

**ミッション**: 駅をお気に入りに追加してください。

* **難易度**: ★4
* **正解場所**: `p3` Main Station（駅アイコン 🚉）
* **クリア条件**:
  1. 地図上の駅（Main Station）をタップして詳細パネルを表示する
  2. お気に入りボタン（⭐）をタップしてお気に入りに追加する
* **不正解条件**:
  * 駅以外の場所のお気に入りボタンをタップ → 不正解（ピンタップして詳細パネルを表示するだけはOK）
  * 正解の駅のお気に入りを解除しても不正解にはならない（状態の更新のみ）
* **Insight**: ★（スター）マークはお気に入り登録の国際標準記号。地図・SNS・ECサイトで広く使われており、グレー→黄色への色変化がお気に入り追加の完了を示す。
