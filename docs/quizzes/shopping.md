## 概要

| 項目 | 内容 |
|------|------|
| パッケージ | `packages/quizzes/shopping` |
| クイズ総数 | 4問 |
| テーマ | グローバルECサイト（Amazon風） |
| 特徴 | UnreadableText（難読化テキスト）による言語非依存UI |

### Quiz 1: 水購入クイズ (Water Quiz)

| 項目 | 内容 |
|------|------|
| 難易度 | ★1 |
| ミッション | 水を2つ購入してください |
| 時間制限 | 60秒 |
| タイプ | 商品探索型 |

**クリア条件**
- `water_` で始まるアイテムをカートに合計2個以上入れる
- 購入手続きを完了する（isPurchased = true）

**失敗メッセージ**
- 「水が{個数}個しかありません。2個以上必要です」

**Insight（学習ポイント）**
- アイコンは言語を超える
- 色が行動を促す（黄色いボタン）
- UIパターンが直感を作る

---

### Quiz 2: 購入手続きクイズ (Checkout Quiz)

| 項目 | 内容 |
|------|------|
| 難易度 | ★2 |
| ミッション | 購入手続きを完了してください |
| 時間制限 | 90秒 |
| タイプ | フロー完了型（3ステップ） |

**クリア条件（3ステップすべて完了）**

| ステップ | 内容 | 状態フラグ |
|----------|------|------------|
| 1 | 住所を入力 | isAddressEntered |
| 2 | 支払い方法を選択 | isPaymentSelected |
| 3 | 注文確認チェックボックスをチェック | isConfirmed |

**支払い方法の選択肢**
- クレジットカード
- コンビニ払い
- 代金引換

**ダミー注文サマリー**
- 商品合計: 780円 / 配送料: 0円 / 注文合計: 780円

---

### Quiz 3: 再注文クイズ (Reorder Quiz)

| 項目 | 内容 |
|------|------|
| 難易度 | ★3 |
| ミッション | 直近で注文した商品をもう一度購入してください |
| 時間制限 | 90秒 |
| タイプ | 再検索型 |

**クリア条件**
- ターゲット商品（`snack_chips` / ポテトチップス）をカートに1個以上入れる
- 購入手続きを完了する（isPurchased = true）

**UI特徴**
- 注文履歴セクション（「最近の注文」「3日前に注文」）
- 「もう一度買う」ボタン

**Insight（学習ポイント）**
- 商品画像が記憶のトリガー
- 価格が確信を与える
- 「もう一度買う」ボタンが体験を完結させる

---

### Quiz 4: カート合計金額クイズ (Cart Quiz) 🔥ゲキムズ

| 項目 | 内容 |
|------|------|
| 難易度 | ★5 🔥 |
| ミッション | このカートの合計金額を選んでください |
| 時間制限 | 60秒 |
| タイプ | 計算型（4択） |

**カート内容**

| 商品 | 単価 | 数量 | 小計 |
|------|------|------|------|
| 水 500ml 💧 | 100円 | 3個 | 300円 |
| 茶 500ml 🍵 | 150円 | 2個 | 300円 |
| コーヒー 500ml ☕ | 180円 | 1個 | 180円 |
| **合計** | | | **780円** |

**選択肢**: 580 / 630 / 750 / 780（正解: 780）

**ヒント**
- 「各商品の単価 × 数量を足すと合計が求まります」
