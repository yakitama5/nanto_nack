# 💬 チャット・メッセージカテゴリ クイズ一覧

「文字が読めないチャットアプリ」を舞台に、コミュニケーションツールのUIセオリーを問うクイズシリーズ。

## 概要

| 項目 | 内容 |
|------|------|
| パッケージ | `packages/quizzes/chat` |
| クイズ総数 | 4問 |
| テーマ | メッセージアプリ（LINE / iMessage風） |
| 特徴 | 吹き出しの色・配置、アイコンの意味、ジェスチャー操作の理解 |

---

## クイズ一覧

| # | クイズ名 | 難易度 | 時間制限 |
|---|----------|--------|---------|
| 1 | スタンプ送信クイズ | ★1 | 30秒 |
| 2 | リアクションクイズ | ★2 | 45秒 |
| 3 | 画像送信クイズ | ★3 | 60秒 |
| 4 | メッセージ取消クイズ | ★5 🔥 | 60秒 |

---

### Quiz 1: スタンプ送信クイズ
**ミッション**: 相手にスタンプを1つ送信してください。
* **難易度**: ★1
* **クリア条件**: 入力欄の横にある「顔文字（☻）」アイコンをタップし、下部に出現したスタンプ一覧の中から任意の画像を1つ選択して、タイムラインに送信すること。
* **Insight**: 「顔文字アイコン＝スタンプ/絵文字の呼び出し」という推測と、スタンプはテキスト入力とは別の専用UIから送るというチャットアプリの基本セオリーを問う。

### Quiz 2: リアクションクイズ
**ミッション**: 相手の最新のメッセージに「いいね（❤️）」をしてください。
* **難易度**: ★2
* **クリア条件**: 左側（相手）の最新の吹き出し横にある顔文字アイコンをタップし、赤色のハートアイコンに変化させること。

### Quiz 3: 画像送信クイズ
**ミッション**: カメラロールから写真を1枚送信してください。
* **難易度**: ★3
* **クリア条件**: 入力欄左の「＋」または「📷」をタップ → ボトムシートから画像を1つ選択 → 送信ボタンを押すこと。

### Quiz 4: メッセージ取消クイズ 🔥ゲキムズ
**ミッション**: 自分が最後に送ってしまったメッセージ（誤爆）を取り消してください。
* **難易度**: ★5 🔥
* **クリア条件**: 自分の最新の吹き出しを**長押し（Long Press）**し、出現したメニューから**一番下にある赤文字の項目**（送信取消）を選択すること。
* **Insight**: 「隠されたメニュー」は長押しで出ること、そして「破壊的アクション（削除・取消）は赤色で表現される」という強烈なUIセオリーを問う。
